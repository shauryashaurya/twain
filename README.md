# twain
A comprehensive typing tutor - named for **Mark Twain**, who was the first author to submit a fully typewritten manuscript for publishing.   

Update: In my travels and searches, it seems like that fact about Mark Twain is maybe semi-untrue, but it's a good name for a typing app, so am keeping it. Smiley face :)
  
Run offline with `npm run dev`  
or check out the online version here: [https://shauryashaurya.github.io/twain/](https://shauryashaurya.github.io/twain/)
